## 💡 핵심 발견

### 1. 시스템은 작동한다
- 블라인드 상태에서도 정확한 저수지 감지
- AI/Power/Energy 힌트 없이 자동 분류
- 순서와 강도까지 일치

### 2. 기업신호 vs 정책신호
- 기업 투자: Lead Time 정확 (7개월)
- 정부 정책: Lead Time 불안정 (5-13개월)
- **신호 종류에 따라 가중치 달라짐**


## 🎓 학습 포인트

### 데이터 정확성의 중요성
```
❌ 224,420건 DART 공시 (잘못된 필터)
  → 투자와 무관한 금융상품 공시

✅ 28건 실제 투자 사건 (올바른 선택)
  → 정확한 저수지 감지
  
결론: "큰 데이터" ≠ "좋은 데이터"
```

### 블라인드의 중요성
```
후견지명 있을 때:
  "AI, Power, Energy를 찾자" → 찾은 것처럼 보임

후견지명 없을 때:
  "뭐가 저수지인지 찾아" → 정확하게 맞음

결론: 검증의 신뢰성은 "눈가림"에서 나온다
```

### 신호 종류의 차이
```
기업 투자 신호:
  - 신뢰도: 높음
  - Lead Time: 정확 (7개월)
  - 사용: 주신호

정부 정책 신호:
  - 신뢰도: 중간
  - Lead Time: 불안정 (5-13개월)
  - 사용: 보조신호

결론: 신호를 다르게 가중치 줘야 한다
```
