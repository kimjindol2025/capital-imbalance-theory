### 📌 중요 발견

```
224,420건 (전체 공시)
    ↓
  211건 (투자 관련) = 0.09%
    ↓
  187건 (무작위 샘플)

→ 99.91%는 투자와 무관 (자기주식, 지분공시, 임원 정보 등)
→ 실제 투자 신호는 매우 드물게 나타남
→ 따라서 센서 수집이 얼마나 중요한지 증명됨
```

### 샘플링 방식

```
❌ Before: items[:50] (앞에서 자르기)
   → 시간 편향 발생 (최신 공시만)

✅ After: random.sample() (무작위 샘플링)
   → 편향 제거, 대표성 확보
```
